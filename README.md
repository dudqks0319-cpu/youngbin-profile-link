# 🔗 영빈 프로필 링크

[![Live Demo](https://img.shields.io/badge/🌐_Live_Demo-Visit_Site-8B5CF6?style=for-the-badge)](https://youngbinlink-hafxukkj.manus.space/)
[![Admin Panel](https://img.shields.io/badge/🔐_Admin-Dashboard-3B82F6?style=for-the-badge)](https://youngbinlink-hafxukkj.manus.space/admin)

> 육아용품 크리에이터 영빈을 위한 프로필 링크 페이지

## ✨ 주요 기능

### 🌐 공개 페이지
- 📱 모바일 최적화 반응형 디자인
- 🌙 다크모드/라이트모드 전환
- ⏰ 시간대별 환영 메시지
- 🎬 부드러운 애니메이션 효과
- 🔗 스마트 링크 (우선순위/일반 링크)
- 🖼️ 추천 게시물 캐러셀
- 🛍️ 제품 갤러리 (쿠팡파트너스 연동)
- 📧 뉴스레터 구독

### 🔐 관리자 대시보드
- 📊 링크 클릭 분석 통계
- ✏️ 프로필 정보 편집
- 🔗 링크 관리 (드래그앤드롭 순서 변경)
- 🖼️ 캐러셀 이미지 관리
- 🛍️ 제품 갤러리 관리

## 🛠️ 기술 스택

| 분류 | 기술 |
|------|------|
| Frontend | React 19, TypeScript, TailwindCSS 4, Framer Motion |
| Backend | Express, tRPC |
| Database | MySQL (Drizzle ORM) |
| Styling | Radix UI, Lucide Icons |

## 🚀 빠른 시작

```bash
# 의존성 설치
pnpm install

# 개발 서버 실행
pnpm dev

# 빌드
pnpm build
```

## 📁 프로젝트 구조

```
youngbin-profile-link/
├── client/                 # 프론트엔드
│   └── src/
│       ├── components/     # UI 컴포넌트
│       ├── pages/          # 페이지 컴포넌트
│       └── contexts/       # React Context
├── server/                 # 백엔드 API
├── shared/                 # 공유 타입/유틸
└── drizzle/               # DB 마이그레이션
```
